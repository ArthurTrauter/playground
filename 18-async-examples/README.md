# async-examples [![NPM version][npm-image]][npm-url] [![Build Status][travis-image]][travis-url] [![Dependency Status][daviddm-image]][daviddm-url]

## Install

```sh
$ npm install --save async-examples
```


## Usage

```js

asyncExamples('Rainbow');
```
